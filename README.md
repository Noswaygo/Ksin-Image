# Ksin Image

一个功能完善的桌面图片管理应用

## 功能特性

- 🔐 **多种登录方式**: 支持用户名、邮箱、手机号登录/注册
- 🖼️ **图片管理**: 上传、查看、编辑、删除图片
- 📁 **相册管理**: 创建、管理图片相册
- 🔗 **分享功能**: 创建图片和相册分享
- 💳 **订阅系统**: 购买和管理订阅套餐
- 📋 **订单管理**: 查看和管理订单
- 🎫 **工单系统**: 提交和管理工单
- 🔑 **Token管理**: 创建和管理API令牌
- 👤 **个人资料**: 管理个人信息
- ⚙️ **系统设置**: 丰富的设置选项
- 🌓 **主题切换**: 支持亮色/暗色模式
- 🎨 **主题颜色**: 可自定义主题颜色
- 🌍 **多语言**: 支持中英文切换
- 🔄 **自动更新**: 支持在线更新
- 📌 **系统托盘**: 托盘图标和菜单

## 技术栈

- **前端框架**: Vue 3 + Vite
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router
- **国际化**: Vue I18n
- **桌面应用**: Electron
- **HTTP客户端**: Axios

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
# 启动Vite开发服务器
npm run dev

# 启动Electron开发模式
npm run electron:dev
```

## 构建打包

```bash
npm run build
```

构建完成后,安装包将生成在 `dist` 目录中。

## 项目结构

```

├── electron/              # Electron主进程
│   ├── main.js           # 主进程入口

│   ├── assets/          # 静态资源
│   ├── components/      # 公共组件
│   ├── i18n/           # 国际化配置
│   ├── router/         # 路由配置
│   ├── stores/         # 状态管理
│   ├── styles/         # 样式文件
│   ├── utils/          # 工具函数
│   ├── views/          # 页面组件

```

## 页面说明

### 登录/注册
- 支持用户名、邮箱、手机号三种登录方式
- 集成图形验证码
- OAuth第三方登录

### 广场
- 浏览热门图片和相册
- 点赞和举报功能
- 热门标签

### 我的图片
- 查看所有上传的图片
- 批量操作
- 复制链接、下载

### 我的相册
- 创建和管理相册
- 添加/移除图片
- 设置封面

### 上传图片
- 拖拽上传
- 批量上传
- 选择相册和标签

### 我的分享
- 创建分享链接
- 设置访问密码和过期时间
- 查看分享统计

### 购买订阅
- 查看当前订阅
- 浏览套餐
- 购买订阅

### 我的订单
- 订单列表
- 订单详情
- 支付订单

### 我的工单
- 提交工单
- 工单回复
- 关闭工单

### 我的令牌
- 创建API令牌
- 管理令牌权限
- 查看令牌信息

### 个人资料
- 修改个人信息
- 修改密码
- 绑定/换绑手机号和邮箱
- 设置头像和背景

### 设置
- 外观设置(主题、颜色、语言)
- 上传设置
- 窗口行为(关闭、最小化)
- 系统设置(开机启动、通知、自动更新)
- 关于信息

## 系统托盘

应用支持系统托盘功能:
- 双击托盘图标显示/隐藏窗口
- 右键菜单:显示主窗口、检查更新、退出

## 窗口行为

设置中可配置:
- 关闭窗口时:最小化到托盘或直接退出
- 最小化时:最小化到任务栏或托盘

## 主题系统

- 亮色/暗色模式切换
- 8种预设主题颜色
- 所有Element Plus组件适配暗色主题
